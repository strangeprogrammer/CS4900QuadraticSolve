**fp -> directory info:**

fp (floating point) directory holds implementation and test files which are listed below.
All of the implementation files in this directory are used for validation, as well as some calculations for quadsolver.


Implementation files:
	- ieee.c, ieee.h
	- quadfunc.c, quadfunc.h
	- validate.c, validate.h

Test files:
	- mock_validate_deps.c
	- t_isq_feq.c
	- t_qfunc.c
	- t_valid.c
