# CS4900QuadraticSolve
Solve the quadratic formula given inputs 'a', 'b', and 'c'.  
The amount of x intercepts(0-2) is printed out. the values of the intercepts are also printed out if any exist.

The directory "src" contains all files used for compiling and executing quadsolver. Also unit testing is in "src", each different test is in the folder with the module it is testing.

The directory "documentation" has all files containing instructions and information user may need to install, compile, run, use, maintain, or test this program.
